JAVA ANDROID STUDIO APPLICATION	

The application is an inventory management system, a credentials manager, and rental service with built-in 
authorization verification. The authorization level determines the functions that can be accessed by the user.
The application allows creation, modification, and deletion of inventory objects. The application will use objects
for the rental service. The full project description is included in project lifecycle document. The document is 
created from the prespective on the project manager from project proposal to client acceptance of the project.
